  Konsolidasi Pengadaan : Subsystem

Paket pemaketan berfungsi untuk mendefinisikan paket pengadaan. Sebuah paket dapat berasal dari 1 RUP, banyak RUP, atau 1 RUP dipecah menjadi beberapa paket.

 

Sebuah Paket Pengadaan mungkin mengalami pemecahan Paket Pengadaan, atau Banyak Paket Pengadaan mungkin mengalami penggabungan Paket Pengadaan. Kegiatan pemecahan atau penggabungan paket disebut dengan kegiatan Konsolidasi Pengadaan agar pengadaan efisien dan efektif. Catatan : Aktor KUKPBJ dan perannya tidak ada dalam Perek No 2 Tahun 2023, sehingga Flow Map yang diberikan oleh Pak Muchid perlu disesuaikan.
